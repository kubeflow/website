## Initial setup for your existing cluster

Get the Kubeconfig file:

	```shell
	ibmcloud ks cluster config --cluster $CLUSTER_NAME
	```

From here on, please see [Install Kubeflow](/docs/ibm/deploy/install-kubeflow).
